# Smart-CrossRef
Smart Cross Referencing Pane Add-In for Microsoft Word
An intelligent, high-performance VSTO Custom Task Pane for Microsoft Word. It dynamically monitors your cursor position to suggest context-aware cross-reference targets (Headings, Figures, Tables, Bookmarks) within a 3-page localized radius. It can fully replace the existing old cross-reference dialog in MS Word.
This is theme aware and built with WPF UI with AI support.

* **Status:** Active / Production-Ready
* **Tech Stack:** C#, WPF, VSTO, Word Interop
